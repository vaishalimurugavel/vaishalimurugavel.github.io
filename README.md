# Vaishali Murugavel - Portfolio Website

A professional portfolio website showcasing my experience as a Software Developer and Computer Science graduate student.

## 🚀 Live Demo

Visit the live website: [https://vaishalimurugavel.github.io](https://vaishalimurugavel.github.io/)

## 🛠️ Built With

- **React** - Frontend framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Vite** - Build tool
- **Lucide React** - Icons

## 📋 Features

- **Responsive Design** - Works on all devices
- **Modern UI** - Clean and professional design
- **Smooth Navigation** - Seamless scrolling between sections
- **Interactive Elements** - Hover effects and animations

## 🏗️ Sections

- **Hero** - Introduction and call-to-action
- **About** - Professional summary and skills
- **Education** - Academic background
- **Experience** - Work history and achievements
- **Projects** - Technical projects and accomplishments
- **Contact** - Get in touch information

## 🚀 Local Development

1. Clone the repository:
```bash
git clone https://github.com/yourusername/vaishali-portfolio.git
cd vaishali-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📦 Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions. Any push to the `main` branch will trigger a new deployment.
